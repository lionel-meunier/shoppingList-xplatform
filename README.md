# On the road to Progressive Web Apps: building real-cross platform Angular apps with Cordova & Electron

Presented at ng-europe 2018, Paris.

## Setup

```
$ npm in
```

## Start PWA build

```
$ npm run start-prod
```

## Start Cordova build

```
$ npm run build-mobile
```

## Start Electron build

```
$ npm run build-desktop
```
